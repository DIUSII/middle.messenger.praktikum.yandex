# Описание
Пет-проект месенджера

# Установка
Установка зависимостей
```
npm i && npm i -g serve
```
```
yarn && yarn global add serve
```
Запуск проекта
```
npm run dev
```
```
yarn dev
```
Билд проекта
```
npm run build
```
```
yarn build
```
Сборка и запуск проекта
```
npm run start
```
```
yarn start
```
